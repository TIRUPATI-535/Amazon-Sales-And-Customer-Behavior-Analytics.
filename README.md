# 📊 Amazon Sales And Customer Behavior Analytics.

## 🚀 Project Overview

This project focuses on analyzing Amazon e-commerce data to understand **sales performance, customer behavior, and pricing strategies** using **Python, SQL, and Power BI**.

The goal is to extract meaningful insights from raw data and present them through an interactive dashboard for better business decision-making.

---

## 🧰 Tools & Technologies Used

* 🐍 Python (Pandas,) - For Data Cleaning & Analysis (EDA)
* 🗄️ MySQL (For Business Insights)
* 📊 Power BI -(Dashboard)
* 📁 CSV Dataset (Amazon Data)

---

## 📂 Dataset Description

The dataset contains information related to:

* Orders & Transactions
* Products & Categories
* Customers & Locations
* Pricing & Discounts

### 📌 Key Columns:

* OrderID, OrderDate
* CustomerName, City, State, Country
* ProductName, Category,
* Quantity, UnitPrice, Discount
* ShippingCost, TotalAmount
* PaymentMethod, OrderStatus

---

## 🐍 Python (Data Cleaning & Processing)

Performed data preprocessing using Pandas:

* Handled missing values
* Removed duplicates
* Converted data types
* Created new features:

  * Sales = Quantity × UnitPrice
  * Customer Segmentation
  * Date-based columns (Year, Month)

---

## 🗄️ SQL (Data Analysis)

Used SQL for business insights:

* Total Revenue & Orders
* Top Products & Categories
* Customer-wise spending
* Payment method analysis
* Window functions:

  * RANK()
  * DENSE_RANK()
* CTE and subqueries

---

## 📊 Power BI Dashboard

Created an interactive dashboard with multiple pages:

### 📌 Overview

* Total Revenue, Orders
* Monthly Sales Trend

### 📌 Product Analysis

* Top-performing products
* Category-wise sales

### 📌 Customer Behavior

* High-value customers
* Location-based insights

### 📌 Pricing Analysis

* Discount impact
* Shipping cost trends

## 💡 Key Insights

* Identified top revenue-generating products and categories
